Title
EduPortal: Professional Student Grade & Course Planner

Description
A comprehensive, full-stack academic management platform designed to streamline student progress and administrative oversight. EduPortal features an AI-powered Study Planner for personalized scheduling, automated GPA & Grade Tracking, and a strategic Course Planner that compares optimization algorithms (Greedy vs. Dynamic Programming).

Built with a professional light-themed interface, the platform supports role-based authentication (Student/Admin), searchable course directories, and interactive performance leaderboards, providing a "Proper Website" experience for modern academic environments.
